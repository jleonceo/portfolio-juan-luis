### 📈 Análisis Contable — TechAcces SL
**`/analisis_contable`**

Análisis financiero completo sobre base de datos MySQL real (17.000+ registros contables):

- Propuesta de medidas con impacto de +148.006€ en resultado
- Exportación automatizada a Excel (5 hojas)

*Stack: Python, Pandas, SQLAlchemy, Matplotlib, MySQL*

### 🎯 Lead Scoring con Machine Learning
**`/lead_scoring`**

Clasificador para priorizar leads con mayor probabilidad de compra:
- Dataset: 9.093 registros, 21 variables, tasa de conversión 37,6%
- Modelo ganador: XGBoost con AUC-ROC **0,98** y F1-Score **0,93**
- Hallazgo clave: fuentes de referencia con 91,8% de conversión vs 26% en redes sociales

*Stack: Python, Scikit-learn, XGBoost, Pandas, Seaborn*
